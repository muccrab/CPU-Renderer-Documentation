Detailed Description

Abstract base class for shader programs.

Generic shader program implementation using specified vertex and fragment shader types.

Template Parameters
TVERThe type of the vertex shader, must implement IVertexShader and have a parameterless constructor.
TFRAGThe type of the fragment shader, must implement IFragmentShader and have a parameterless constructor.
Type Constraints
TVER :IVertexShader 
TVER :new() 
TFRAG :IFragmentShader 
TFRAG :new()
